项目名称：全国大健康专家智能信息平台

愿景：成为澳大利亚大健康领域的国家级专家数据与智能协作基础设施

## **1. 项目背景（Problem）**

* 大健康行业专家信息分散、更新慢、可信度难验证
* 医疗机构、科研单位、企业无法快速找到合适的专家合作
* 信息不透明导致资源浪费、协作效率低、商业机会被错失
* 行业迫切需要权威级、智能化、可扩展的专家信息平台

## **2. 核心目标（Our Mission）**

* 建立全国统一的专家数据中心
* 基于 AI 的智能检索、匹配与推荐
* 支持专家合作、科研对接、商业伙伴发现
* 成为大健康产业的数字底座与行业高价值数据资产

## **3. 核心提升点（Key Value Propositions）**

* 权威可信：专家档案数据结构化、持续更新、可验证
* 智能推荐：AI 驱动的意图识别、画像、匹配与预测
* 高级分析：专家竞争力报告、行业趋势洞察与商业价值分析
* 多模态 AI：支持自然语言、图表、文件、语音的智能问答
* 高性能架构：可支撑国家级专家数量与高并发访问量

## **4. 目标用户与市场规模**

**主要用户群体：**

* 医疗机构、诊所、医院集团
* 药企、生物科技公司
* 健康管理机构、康养中心
* 高校、研究机构
* 政府与行业协会

**市场规模：**

* 澳大利亚大健康产业规模 >1,900 亿澳元
* 信息与数字化服务市场 >100 亿澳元
* 平台级数据资产价值长期增长空间巨大

## **5. OZHG v1.0 建设完成**

* 专家信息管理与标签体系
* 智能搜索与高级过滤
* 专家推荐（基于领域、项目需求、研究方向）
* 专家合作对接与内部工作流管理
* 多角色权限系统（管理员、专家、访客、机构）
* 内容管理：新闻、研究成果、项目动态

## **7. 技术架构与部署（可支撑全国规模）**

* **Web 前端**：Next.js + Vercel（高性能、低维护成本）
* **数据库与身份验证**：Supabase（Postgres + Auth + Storage）
* **AI 服务与模型推理**：Azure AI / OpenAI 服务
* **权限与审计**：可满足医疗行业的合规要求
* **可扩展架构**：多区域部署、可随专家与机构增长线性扩展

| 模块                   | 平台                                    | 说明                     |
| -------------------- | ------------------------------------- | ---------------------- |
| Web 前端               | **Vercel**                            | Next.js 最佳部署平台，全球加速    |
| API / Edge Functions | **Vercel Functions**                  | 轻量后端逻辑（推荐理由生成等）        |
| 数据库                  | **Supabase PostgreSQL**               | 实时数据库 + 向量扩展（pgvector） |
| 身份验证                 | **Supabase Auth**                     | 支持邮箱、社交登录、OTP          |
| 向量搜索                 | **Supabase pgvector**                 | 精准 + 成本低               |
| 文件存储                 | **Supabase Storage**                  | 专家头像/成果文件上传            |
| 分析                   | **Supabase Analytics / BigQuery（可选）** | 行为分析 & BI              |
| AI 模型调用              | **OpenAI API / Azure OpenAI（可混合）**    | 模型推理 + 多模态             |


## **8. 商业模式（Business Model）**

1. **机构会员订阅制**

   * 医院、公司、高校按年付费
   * 提供高级搜索、报告生成、专家推荐、协作工具

2. **企业需求撮合抽佣**

   * 企业发布项目 → 专家合作 → 平台抽佣

3. **专家品牌增值服务**

   * 竞争力分析报告
   * 个人主页定制、学术影响力可视化

4. **行业数据与趋势报告**

   * 产业研究机构订阅

* 可快速规模化、可持续扩张
